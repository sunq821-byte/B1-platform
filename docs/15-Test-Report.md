## 1 测试概述

验证 B1 智慧实训平台各功能模块的正确性、完整性和可用性，确保系统达到比赛交付标准。

### 1.2 测试范围

覆盖 3 角色（学生 / 教师 / 管理员）、24 个页面、10 个核心业务功能的完整功能测试。

|---|---|

| 数据库 | MySQL 8.0 (lcr.loongnix.cn 镜像) |
| 缓存 | Redis 7 (lcr.loongnix.cn 镜像) |

| 测试客户端 | Chrome 浏览器，通过 HTTP 访问 |
