## 1 产品简介

B1 智慧实训平台是面向高校软件类专业实训教学全流程的智能化管理与评价平台。系统覆盖"课程管理 → 实训发布 → 标准配置 → 成果提交 → 智能分析 → 教师审核 → 能力成长 → 教学复盘"完整业务闭环，以 AI 大模型辅助教学评价，降低教师批改工作量 60% 以上。

### 1.1 核心价值

- **教师**：AI 自动初评 + 人工复核，大幅减少重复性批改工作
- **学生**：提交后即时获得 AI 诊断反馈，明确改进方向
- **教研**：多维度学情数据驱动课程迭代与教学改革
- **管理**：实训教学全流程数字化，满足归档与督导要求

### 1.2 技术架构

前后端分离架构：Vue 3 + TypeScript 前端，Spring Boot 3.5 + JDK 21 后端，MySQL + Redis + MinIO 数据存储，集成 DeepSeek / 通义千问大模型。

## 2 用户角色

系统支持三类角色，权限隔离，登录后自动匹配对应功能页面。

| 角色 | 权限范围 |
|---|---|
| **学生** | 课程查看、任务接收、成果提交、成绩查看、个人报告、成长中心 |
| **教师** | 课程管理、任务发布、AI 诊断查看、学生复核评分、报表统计、学生管理 |
| **管理员** | 用户管理、班级管理、系统配置、操作日志、系统监控 |

## 3 公共功能

### 3.1 登录与鉴权

系统采用基于 Sa-Token 的 JWT 认证机制，支持统一登录页面。登录后根据角色自动跳转对应仪表盘。

**登录页面**：输入用户名和密码，系统校验后生成 Token 并写入请求头，后续请求自动携带。

![登录页面](../prototypes/screenshots/01-login.png)

### 3.2 导航系统

顶部导航栏包含系统 Logo、主导航菜单（按角色动态展示）、通知铃铛、用户头像下拉菜单（个人中心 / 修改密码 / 退出登录）。

### 3.3 通用交互规范

- 数据表格支持分页、排序、筛选
- 表单提交后即时反馈（成功 Toast / 错误提示）
- 加载状态展示骨架屏或 Loading 动画
- 空数据展示 Empty State 引导文案
- 错误场景展示 Error State 与重试按钮
- 403/404 页面提供友好导航

## 4 学生端

学生端共 7 个功能页面，覆盖从课程查看到个人成长的全流程。

### 4.1 学生仪表盘

登录后首页，卡片展示关键数据：课程数量、待提交任务数、已批阅成绩数、AI 评测平均分。下方展示近期任务列表和课程概览。

![学生仪表盘](../prototypes/screenshots/02-dashboard-student.png)

### 4.2 课程列表

展示当前学期所有已选课程，包含课程名称、授课教师、进度、任务完成情况。点击进入课程详情。

### 4.3 课程详情

展示单个课程下的全部实训任务列表，含任务名称、截止时间、提交状态（未提交 / 已提交 / 已批阅）、成绩。支持点击进入任务详情。

### 4.4 任务详情

展示任务描述、实训要求、提交方式（文件 / Git URL）、截止时间。已完成提交的展示提交记录和 AI 评分结果。

### 4.5 成果提交

![提交页面](../prototypes/screenshots/07-submission.png)

支持两种提交方式：
- **文件上传**：支持 ZIP / PDF / DOC / 代码文件，拖拽或点击上传，单文件最大 50MB
- **Git URL**：输入仓库地址，系统自动验证仓库可访问性

提交后自动触发 AI 分析（如已配置），学生可实时查看分析进度。

### 4.6 AI 诊断结果

![AI 分析](../prototypes/screenshots/09-ai-analysis.png)

展示 AI 多维度评分卡（雷达图 + 维度得分），包含：
- 各维度得分与满分对比
- 扣分项明细
- 改进建议

### 4.7 成绩查看

![学生成绩](../prototypes/screenshots/08-student-grades.png)

展示各课程实训任务的最终成绩（AI 初评 + 教师复核后），支持按课程筛选。可查看单次实训的详细评分明细。

### 4.8 个人报告

生成学生个人实训档案报表，包含：
- 分数趋势折线图（跨学期对比）
- 能力雷达图（多维度能力分布）
- 实训成果列表

支持导出 PDF 和 Excel。

### 4.9 成长中心

![成长中心](../prototypes/screenshots/10-growth-center.png)

展示学生学习成长轨迹：学期实训分数演变、能力维度变化趋势、学习建议。

## 5 教师端

教师端共 11 个功能页面，覆盖课程管理、任务发布、AI 诊断、成绩复核、报表统计。

### 5.1 教师仪表盘

![教师仪表盘](../prototypes/screenshots/03-dashboard-teacher.png)

概览当前学期教学情况：授课课程数、学生总数、待批阅提交数、平均完成率。展示近期提交列表和课程分布。

### 5.2 课程管理

![课程管理](../prototypes/screenshots/13-course-management.png)

支持创建、编辑、归档课程。课程信息包括：课程名称、课程代码、学期、学分、课程描述。支持为课程关联学生和配置评价标准。

### 5.3 实训任务管理

![实训管理](../prototypes/screenshots/14-training-management.png)

在课程下创建和管理实训任务：
- 任务名称、描述、实训要求
- 提交方式（文件上传 / Git URL / 混合）
- 截止时间、最大提交次数
- 评分标准关联
- 批量发布到多个班级

### 5.4 学生管理

查看和管理课程下的学生列表，查看学生提交状态、成绩概况。支持搜索和筛选。

### 5.5 提交审核

![教师评审](../prototypes/screenshots/11-teacher-review.png)

查看学生提交的实训成果，包含：
- **代码文件预览**：在线查看 ZIP 包内代码文件
- **AI 诊断结果**：查看 AI 对本次提交的多维度分析
- **人工复核**：对 AI 评分进行采纳/调整，添加教师评语
- **终审操作**：通过 / 驳回 / 退回重提

### 5.6 催办提醒

对未提交任务的学生发送催办通知，支持单选和批量催办。被催办学生会在通知铃铛中收到提醒。

### 5.7 报表中心

![报表中心](../prototypes/screenshots/12-report-center.png)

- **班级学情报表**：班级整体成绩分布、任务完成率、分数段统计
- **学院教学质量报表**：跨班级对比分析、学期趋势
- **跨班对比**：柱状图对比不同班级同一任务表现

支持导出 PDF 和 Excel 格式。

### 5.8 评价标准管理

![评价标准](../prototypes/screenshots/17-evaluation-standards.png)

定义评价维度与权重配置，支持模板库功能（预置模板一键复制复用）。系统固定四维评价框架（完整性 / 规范性 / 正确性 / 创新性），教师可为每个任务配置自定义评分细则。

## 7 AI 功能说明

### 7.1 AI 自动评分

学生提交实训成果后，系统自动调用大模型进行多维度分析评分：

- **代码规范性检测**：语法规范、命名规范、代码格式
- **功能完整性校验**：需求覆盖率、核心功能实现
- **创新性评估**：方案的创新程度和扩展性
- **报告质量**：文档完整性和规范性

### 7.2 流式输出

AI 分析过程通过 SSE（Server-Sent Events）实时推送，前端以打字机动画展示分析进度和结果。

### 7.3 AI 限制与安全保障

- AI 仅提供建议评分，最终分数由教师确认
- 所有 AI 输出可追溯、可复核
- 支持超时降级：AI 调用超时自动返回降级结果并提示
- AI 密钥不入库，通过环境变量注入
